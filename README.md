# BattleNet-Login-Helper

战网多账号管理与自动登录助手

![](./imgs/prev.png)

## 功能

- 存储账号信息, 包括账号密码与登录信息
- 一键登录选中账号至对应地区战网

## 使用

### 应用程序

前往release界面下载使用

### 源码运行

需要安装`python`, 推荐使用`3.10.8`

```shell
# 克隆源码并安装依赖
git clone https://github.com/WitchElaina/BattleNet-Login-Helper.git
cd BattleNet-Login-Helper
pip install -r requirements.txt

# 运行程序
python main.py
```
